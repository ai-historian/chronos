Analyze a single scanned page using a vision model. Sends the page image together with a prompt to a specialist model and returns the text response. Use this when you need to understand the content of a page. After this call, use follow_up_question to ask the same page expert follow-up questions without re-sending the image. page_id: the page number (e.g. 1 for page_0001.png). prompt: what to ask about the page. model: which Gemini model to use (default: gemini-3-flash-preview). output_file: if provided, write the response to this file in the source directory and return a short confirmation instead of the full text.
